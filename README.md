# Werbos

Werbos is a small web application for practicing the 100 most common Spanish verbs.

The first version is planned as a vanilla HTML, CSS, and JavaScript PWA. Exercise content should be delivered as downloadable JSON resources, and progress should be stored on the user's device without accounts or login.

## Foundation Documents

- [Werbos Constitution](docs/constitution.md) defines the product and engineering principles.
- [Initial PWA Architecture](docs/initial-pwa-architecture.md) describes the first version's structure, data model, offline behavior, and GitHub Pages deployment assumptions.
- [Tailwind CSS Decision](docs/tailwind-css-decision.md) records the initial styling recommendation.

## First-Version Goals

- Practice the 100 most common Spanish verbs.
- Work as an installable PWA.
- Load exercise data from static JSON files.
- Store progress locally on each device.
- Avoid login, accounts, backend services, and server-side state.
- Deploy as a static site on GitHub Pages.
