# Werbos Design System Proposal

This folder contains the Practice Deck proposal for the Werbos design system.
It is a mobile-first static artifact that uses Tailwind CSS from the CDN while
the product direction is still being explored.

## Contents

- `index.html` - component board for the main practice and quiz states.

## Direction

Practice Deck treats each exercise like a focused study card with clear progress,
large touch targets, and slightly brighter reinforcement. Placeholder marks are
included for the Werbos logo, Zorrito, and small UI icons.
