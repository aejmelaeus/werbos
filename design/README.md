# Werbos Design System Proposal

This folder contains the Lesson Rail proposal for the Werbos design system.
It is a static, mobile-first artifact using the Tailwind CSS CDN for early
iteration.

## Contents

- `index.html` - source-of-truth component board for lesson and quiz UI.

## Direction

Lesson Rail favors a calm app workspace: persistent lesson status, compact
grammar context, Zorrito guidance, and answer choices that are optimized for
repeated mobile practice.
